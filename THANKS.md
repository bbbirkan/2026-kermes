# Credits & Inspiration

## Hermes Agent — Nous Research

Kermes was built on top of, and deeply inspired by,
**[Hermes Agent](https://github.com/NousResearch/hermes-agent)** by [Nous Research](https://nousresearch.com).

Hermes solved the genuinely hard problem: multi-platform messaging,
persistent memory (SOUL.md / MEMORY.md / USER.md), a skills system, and
heterogeneous model delegation — all in a single coherent architecture.

We are grateful for that work. Kermes adds nothing to the core agent design.
It adds one thing only: a token optimization layer that decides *which model*
to call and *how much context* to send before Hermes reaches the API.

> "We did not build a new agent.
>  We built a smarter wallet for the agent you already have."

If Kermes is useful to you, please also star the Hermes repository:
https://github.com/NousResearch/hermes-agent

## Name & Logo

The name **Kermes** comes from the *Kermes vermilio* insect,
the historical source of crimson dye in the Mediterranean world.
The English word *crimson* derives from it (via Arabic *qirmiz*,
Turkish *kırmızı*).

The Kermes logo is a merged **K**+**H** letterform:
the **K** of Kermes carrying the **H** of Hermes within its strokes.
The body is rendered in kermes crimson (#C41E3A).
The bottom strokes — the feet of both letters — are amber gold (#FFB800),
representing the cost savings passed to the user.
